# Multi-Agent 議論運用ガイド（暫定）

- 目的: 複数エージェント議論の手順と命名/ログ運用を一箇所にまとめ、毎回のセットアップを簡略化する。
- 想定スコープ: 司会型 + 必要箇所のみ順次収集＋要約統合のハイブリッド運用。

## 使い方

```
/discussion-forum [テーマ]
```

Claude Code で上記コマンドを実行すると、複数ロールによる議論が開始される。

## ドキュメント構成

| ファイル | 説明 |
|----------|------|
| `workflow.md` | 司会プロンプトの7ステップ実運用サイクル |
| `roles.md` | ロールの定義と出力フォーマット |
| `logging.md` | ログのディレクトリ/ファイル命名と保存ルール |
| `summary.template.md` | 要約ファイルテンプレート（YAML フロントマター付き） |
| `evaluation.md` | 複数Agent会議の効果・課題・価値の整理 |
| `prior-art.md` | 先行事例と研究（学術研究、OSSフレームワーク、商用実装） |

## 議論ログの保存先

```
docs/decisions/logs/YYYY-MM-DD-topic/
├── summary.md      # 要約（summary.template.md 形式）
└── NN-role-raw.md  # 生ログ（必要に応じて）
```

詳細は `logging.md` を参照。
